# BoredRoom Sound Credits

Current shipped effects are synthesized in `src/lib/sounds.ts` using the Web Audio API.

No third-party dice or game audio assets are bundled in this build. The Ludo
"dice on glass" cue is an original procedural foley-style effect composed from
filtered noise bursts, high-frequency clacks, and a final settle hit.

If a recorded sample replaces the procedural cue later, add the source URL,
license, attribution text, and modification notes here before shipping.

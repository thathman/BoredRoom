# Contributing

Thank you for improving BoredRoom.

Run the app locally with the commands in `README.md`.

Before sending changes, run lint, typecheck, tests, build, and the relevant smoke checks.

Product rules:

- Keep party, game, round, and vote flows separate.
- Keep gameplay server-authoritative.
- Keep the public display clean.
- Keep host controls on the companion screen where possible.
- Keep controllers focused on player actions.
- Add tests for behavior changes.
- Update docs for new flows or settings.
